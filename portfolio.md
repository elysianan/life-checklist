## 一句话介绍

「人生已完成清单」是一款面向移动端的极简人生记录 Web 应用，帮助用户把想完成的事变成清单、追踪进度、记录人生大事，并把「人生倒计时」的焦虑转化为珍惜余生的动力。

---

## 项目背景

市面上大多数清单/打卡工具只关注「完成率」，容易让人陷入效率焦虑。我希望做一款更温和的工具：

- 它不只记录任务，还记录人生大事（人生轴）
- 它不只显示进度，还显示人生剩余时间（余生闹钟）
- 它不只提供模板，还能基于数据智能推荐（AI 推荐）

这个项目从 v1.0 迭代到 **v6.9.0**，功能逐步扩展，但始终保持着简洁、克制的产品气质。

## 核心功能

### 1. 我的人生清单
- 首页展示所有自定义清单卡片
- 支持长按拖动排序
- 一屏统计概览：总清单 / 已完成 / 总任务
- 顶栏一键创建新清单，操作路径极短

### 2. 清单详情与打卡
- 彩色标题条 + 标签网格布局
- 轻点标签即可完成打卡
- 编辑态支持增删标签、生成分享图
- 连续打卡系统：显示当前连签、最高记录、今日打卡状态

### 3. 清单大全（模板库）
- 8 大类人生清单模板：旅行、电影、阅读、美食、音乐、挑战等
- 中国城市打卡、豆瓣 TOP250、情侣 100 件事等精选内容
- 一键添加到「我的人生进度」

### 4. 人生轴
- 手动记录人生大事记
- 支持精确到月份，同一年多次事件可区分
- 单列 / 双侧时间轴布局切换
- 可导出分享卡片

### 5. 人生进度
- 添加多个人物进度卡片（家人、朋友、宠物等）
- 已活整岁 / 目标寿命 / 可视化进度条
- 「我」的生日与余生闹钟数据同源

### 6. 余生闹钟
- iOS 风格表盘，实时跳动年龄
- 2×2 余生网格：还能看几届世界杯、几个夏天、几个春节……
- 把焦虑转化为「余生还能享受什么」

### 7. AI 能力
- **AI 智能推荐清单**：基于完成数据推荐，支持换一批 / 不感兴趣
- **AI 人生报告**：按月 / 年 / 全部生成阶段总结
- **AI 目标拆解**：输入目标 + 期限，生成 3 阶段可执行路线图
- 所有 AI 功能都有本地规则引擎兜底，无 Key 也能用

## 设计亮点

- **纯原生实现**：没有使用 React/Vue 等框架，完全用原生 JS 管理状态、路由、渲染，锻炼了对浏览器 API 的深入理解。
- **数据持久化**：使用 localStorage 存储用户数据，设计了完整的数据迁移逻辑，保证版本升级不丢数据。
- **AI 降级策略**：所有 AI 功能都有本地规则引擎，真实 API 失败/超时时自动降级，流程不中断。
- **移动端优先**：从交互到视觉都围绕手机场景设计，底部 Tab 导航、滚轮选择器、长按拖拽等。
- **TDD 实践**：核心逻辑（人生轴、日期选择器、人生进度、推荐引擎等）都有单元测试覆盖。
