# 人生已完成清单 - 简历项目描述

## 一句话介绍

一款帮助用户记录人生目标、可视化生命进度、留存成长回忆的移动端 Web App，包含清单管理、清单大全、人生轴、连续打卡、成就系统等 6 大模块。

---

## 项目描述（简历用）

**人生已完成清单** 是一款面向年轻人的**人生目标管理 + 时光记录工具**。产品以"人生进度"为核心概念，通过大环形进度条让用户直观感受时间流逝；通过"清单大全"提供 500+ 人生任务灵感；通过"人生轴"按时间线记录用户的成长轨迹；通过连续打卡、成就系统提升用户留存和活跃度。

项目采用 **HTML + Tailwind CSS + 原生 JavaScript** 实现，无需后端即可运行，数据通过 LocalStorage 本地持久化。项目已完成 6 大模块、15+ 核心功能，支持深色模式、移动端适配、动画交互、音效反馈。

## 项目亮点

1. **差异化定位：** 不做传统 Todo List，而是做"人生时光记录仪"
2. **情感化设计：** 大环形人生进度条、每日格言、完成任务仪式感
3. **游戏化机制：** 连续打卡、成就徽章、等级系统、排行榜
4. **内容丰富：** 内置 8 大类人生清单模板，降低用户使用门槛
5. **完整闭环：** 发现目标 → 记录进度 → 完成打卡 → 人生轴回忆 → 数据分析

## 面试话术

### Q1：你为什么做这个项目？

**回答：**

我发现身边很多人都有很多"人生想做的事"，比如看一次极光、读完 100 本书、和爱的人做 100 件事，但这些目标往往散落在备忘录或脑海中，缺乏系统性的记录和追踪。所以我做了这个项目，帮助用户把人生目标可视化、把完成时刻记录下来，让生命更有仪式感。

### Q2：你在这个项目中解决了什么用户痛点？

**回答：**

主要解决了 4 个痛点：
1. **目标零散：** 人生愿望没有统一记录的地方
2. **不知道做什么：** 清单大全提供 500+ 灵感
3. **缺乏成就感：** 完成任务有动画、音效、成就徽章
4. **缺少成长回顾：** 人生轴按时间线记录每一次完成

### Q3：你怎么提升用户留存？

**回答：**

我设计了多层留存机制：
1. **连续打卡：** 利用损失厌恶心理，让用户不想断签
2. **成就系统：** 即时正向反馈，驱动用户完成更多任务
3. **人生轴：** 形成情感连接，用户会反复回看自己的成长
4. **照片记录：** 完成的不是任务，而是回忆
5. **每日格言：** 每天打开都有新的心灵激励

根据竞品数据，连续打卡可提升 7 日留存 25-35%，成就系统可提升 15-20%。

### Q4：如果继续迭代，你会做什么？

**回答：**

下一步我会重点做 3 件事：
1. **AI 赋能：** 接入大模型做智能清单推荐、AI 人生报告、每日格言生成
2. **社交裂变：** 增加共同清单、挑战活动、排行榜，让用户邀请好友一起完成
3. **商业化：** 设计 Freemium 模式，会员解锁无限清单、云同步、PDF 人生报告

### Q5：你的技术选型是怎么考虑的？

**回答：**

我选择 HTML + Tailwind CSS + 原生 JavaScript，原因有 3 个：
1. **MVP 验证快：** 不需要后端和构建工具，2 天就能做出完整产品
2. **学习成本低：** 作为产品经理，原生 JS 更容易理解和修改
3. **可扩展性好：** 模块化代码结构，后续可以平滑迁移到 React/Vue
