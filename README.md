# 人生已完成清单 ✨

一款极简风格的人生清单 Web App，帮助你记录和追踪人生中想要完成的每一件事。

## 🎯 核心概念

- **首页** - 人生进度：用环形进度条感受时光流逝
- **进度** - 我的人生进度清单：管理所有正在进行的清单
- **大全** - 清单大全：丰富的预设清单模板库
- **人生轴** - 时间线：记录每个完成的精彩瞬间
- **统计** - 数据分析：可视化完成情况
- **我的** - 个人中心：等级、成就、设置

## 🛠️ 技术栈

- **HTML5**
- **Tailwind CSS**
- **原生 JavaScript**
- **Chart.js**（图表）
- **LocalStorage**（数据持久化）

## 🚀 快速开始

1. 下载或克隆项目
2. 双击 `index.html` 文件
3. 在浏览器中打开即可使用

或者使用 VS Code Live Server 扩展。

## ✨ 功能特性

### 首页
- 环形人生进度条
- 每日格言
- 已活天数/剩余天数/今日完成

### 清单大全
- 中国城市打卡
- 豆瓣 TOP250 电影
- 情侣 100 件事
- 人生必尝美食
- 100本高质量必读书单
- 人生必听 100 首歌
- 极限运动挑战
- 人生遗愿清单

### 我的人生进度
- 创建自定义清单
- 添加/编辑/删除任务
- 任务优先级和笔记
- 清单排序

### 人生轴
- 自动记录完成的任务
- 按时间线展示
- 删除记录

### 其他
- 成就徽章系统
- 数据统计图表
- 分享功能
- 深色模式
- 音效反馈
- 移动端适配
- AI 人生报告（本地规则引擎 + 可选大模型）
