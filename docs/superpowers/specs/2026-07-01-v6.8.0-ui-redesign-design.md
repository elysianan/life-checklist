# v6.8.0 UI 视觉重构设计 spec

> **目标**：把人生已完成清单从当前的"多彩渐变卡片风"改造成更接近 Apple 原生极简的界面，解决用户反馈的"UI 土/滞后/大块突兀"问题。
> **范围**：重点改造首页、我的页、底部导航和清单卡片视觉语言；内页(清单/模板/人生轴/人生进度)只同步新视觉语言，不改动业务逻辑。
> **设计方向**：Soft Apple — 白底圆角卡片 + 轻微阴影 + 左侧细色条 + 线型 SVG 图标 + 单一蓝色强调色。

---

## 1. 全局视觉语言

### 1.1 色彩
- **背景**：沿用 Task 2 建立的令牌 `--bg` `#f5f5f7`(浅)/`#000000`(深)。
- **卡片**：`--surface` `#ffffff`/`#1c1c1e`。
- **文字**：`--text` `#1d1d1f`/`#f5f5f7`；`--text-2` `#6e6e73`/`#aeaeb2`；`--text-3` `#a1a1a6`/`#6e6e73`。
- **强调色**：仅使用 `--accent` `#007aff`/`#0a84ff`。
- **删除/成功**：保留 `--danger`、`--success`，但只用于真正的状态/按钮，不作装饰。
- **清单色条**：使用清单自身 `color` 字段，但**饱和度降低 20%** 左右，避免原高饱和 header 的刺眼感。

### 1.2 渐变策略
- **删除几乎所有渐变**：格言卡、连续打卡卡、个人中心渐变头、AI 报告入口渐变、分享卡渐变等。
- **唯一可保留的渐变**：成就徽章解锁态的金色渐变(它承载"勋章"语义),但也可考虑改为纯色金色背景。

### 1.3 阴影与圆角
- **统一圆角**：`16px`(`var(--radius)`)。卡片、按钮、输入框、弹窗全部统一。
- **统一卡片阴影**：`0 1px 3px rgba(0, 0, 0, 0.06)`。
- 需要把 `var(--radius)` 从 `1rem` 固定为 `16px`。

### 1.4 字体层级
| 用途 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 页面标题 | 17px | 600 | `--text` |
| 卡片标题 | 17px | 500 | `--text` |
| 正文/描述 | 15px | 400 | `--text` |
| 辅助/标签 | 13px | 400 | `--text-2` |
| 小标签/徽章 | 11px | 500 | `--accent` 或 `--text-2` |

### 1.5 图标
- 底部导航、顶栏入口、设置项全部使用**2px 线型 SVG 图标**。
- 不使用 emoji 作为功能图标(保留内容性 emoji,如清单卡本身的 emoji)。
- 选中态：icon 保持线型但颜色变为 `--accent`；必要时可切换为 fill 版本。

## 2. 首页重新设计

### 2.1 顶栏
```
[⏰ 线型闹钟图标按钮]  [我的人生清单  17px/600]  [自定义]
```
- 左侧余生闹钟入口：圆形白底按钮，内放细线型闹钟 SVG。
- 标题居中，字号 17px，font-weight 600。
- 右侧"自定义"文字链接，颜色 `--accent`。

### 2.2 今日格言
- **不做大渐变卡片**，改为：
  - 方案 1：白底小卡片，padding `12px 16px`，圆角 16px，轻微阴影。
  - 方案 2：纯文本一行+作者，底部加 1px 分隔线。
- 推荐方案 1(与卡片语言统一)。
- 字体：正文 14px，行高 1.5；作者 12px `--text-2`，右对齐。
- 去掉大灯泡 emoji 或改为小引号线型图标。

### 2.3 统计概览
- 合并到**一张白底卡片**中。
- 横向三等分：总清单 / 已完成 / 总任务。
- 数字 17px font-weight 600；标签 11px `--text-2`。
- 卡片圆角 16px，轻微阴影。

### 2.4 清单卡片
- 白底卡片，圆角 16px，轻微阴影。
- **左侧 4px 圆角色条**，使用清单 `color` 字段(降饱和)。
- 左侧 emoji 缩小到 20px。
- 标题 17px font-weight 500；进度百分比 13px `--text-2` 放在标题下方。
- 卡片间距 10px。
- 删除原来的彩色 header 和百分比右上角显示。

## 3. 底部导航重新设计

- 白底 + 顶部 1px 细线(`--border`)。
- 5 个 tab 全部使用线型 SVG 图标：HOME、清单、人生轴、人生进度、我的。
- 未选中：`--text-2`。
- 选中：`--accent`，可切换为填充版本或保持线型仅变色。
- 标签 10px。
- 移除所有 emoji。

---

## 4. 我的页重新设计

### 4.1 去掉返回键
"我的"是一级 Tab 页面，**不显示返回按钮**。

### 4.2 用户信息区(取代大渐变头)
- 横向一行：头像(48px 灰色占位圆) + 昵称/等级 + 右侧箭头。
- 昵称 "人生探索者" 17px font-weight 600。
- 等级 "Lv.0" 做成小蓝标：背景 `--accent-weak`，文字 `--accent`，11px，圆角 10px。
- 下方小字 "距离下一级还需 1 个任务" 13px `--text-2`。
- **删除**：原渐变 header、大头像、居中标题、等级进度条。

### 4.3 删除 4 宫格统计
- 统计数字已在首页展示，我的页不再重复。
- 需要的数据可通过"统计"Tab 或"人生进度"Tab 查看。

### 4.4 AI 人生报告入口
- 从大渐变卡片改为**设置列表顶部的一项**。
- 左侧 sparkles 线型图标，文字 "AI 人生报告"，右侧 `>`。

### 4.5 最近成就
- 标题改成小标签样式：13px `--text-2`，如 "最近成就"。
- 成就徽章缩小到 32px。
- 空状态提示文字 13px，padding 减少。

### 4.6 设置列表
- 白底卡片分组(类似 iOS 设置)。
- 每组可配一个 12px 大写/淡色标题(中文环境下可用小标签)。
- 每项 padding `10px 16px`，左侧图标 18px 线型，右侧 `>` 或开关。
- "预期寿命"作为最后一项保留在列表中。

## 5. 内页视觉统一(不大改结构)

### 5.1 清单页 / 模板页
- 页面标题 17px/600。
- 分类 chip：白底/浅灰底 + 细边框，选中 `--accent` 背景或边框。
- 模板卡/清单卡与首页清单卡视觉一致：白底 + 左侧色条。
- 去掉模板卡 hover 时的彩色阴影。

### 5.2 人生轴页
- 标题 17px/600。
- 布局切换按钮改成 iOS Segmented Control：圆角胶囊背景，选中 `--accent` 白字。
- 时间轴线条/圆点使用 `--border` 和 `--accent`。
- 事件卡片与全局卡片一致。

### 5.3 人生进度页
- 标题 17px/600。
- 进度卡片白底 + 微阴影 + 16px 圆角。
- 进度条背景 `--border`，前景使用主题色(降饱和)。

### 5.4 余生闹钟 / AI 报告 / 清单详情等二级页
- 保留返回键。
- 统一卡片/按钮/输入框视觉语言。
- 删除渐变，改用纯色或 `--accent-weak`。

## 8. 验证方式

1. **逻辑测试**：`node test/*.cjs` 全部通过。
2. **视觉截图**：`shoot.cjs` 对首页、清单、模板、人生轴、人生进度、我的页、余生闹钟、AI 报告做浅/深双主题截图。
3. **重点检查项**：
   - 底部导航 5 个线型 icon 选中态正确
   - 清单卡左侧色条颜色降饱和后不显脏
   - 首页格言卡片高度适中、不突兀
   - 我的页无返回键、一屏可见主要内容
   - 深色模式无白底漏切、渐变残留

## 9. 风险与注意

- **Tailwind 颜色类冲突**：`index.html` 里仍有很多 `bg-white dark:bg-gray-900` 等 Tailwind 类。需要评估是否替换为令牌类,或保持 class 模式但调整色值。
- **清单卡色条颜色**：降低饱和度后,某些接近灰的颜色可能辨识度下降。实改时可能需要手动调整降饱和算法或给出一个亮度下限。
- **一级页面返回键删除**：要确保这些页面确实只通过底部 Tab 进入,没有其它入口需要返回。
